Accessing your organization's settings

The organization account settings page provides several ways to manage the account, such as billing, team membership, and repository settings.

Only organization owners and billing managers can see and change the billing information and the full set of account settings for an organization. Organization moderators only see moderation settings. For more information, see Roles in an organization.
