About this error

Server error

If the run of a workflow for code scanning fails due to a server error, this may be due to a transient communication issue.

Confirming the cause of the error

You can check the current "Actions" service status on the Status Dashboard.

Fixing the problem

Try running the workflow again. If the problem persists, contact us through the GitHub Support portal.
